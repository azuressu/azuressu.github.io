## 카페 키오스크 만들기

클래스는 총 5개로 구성되어 있습니다.
Main.java, Menu.java, Goods.java, Order.java, MyMenu.java

### 1) Menu.java
```java
public class Menu {
    private String num;     // 메뉴 번호
    private String name;    // 메뉴 이름
    private String detail;  // 메뉴 설명

    // 각 메뉴에 접근 가능한 getter 메소드
    public String getNum() {
        return this.num;
    }
    public String getName() {
        return this.name;
    }
    public String getDetail() {
        return detail;
    }
    // 기본 생성자
    public Menu() {}
    // 생성자
    public Menu(String num, String name, String detail) {
        this.num = num;
        this.name = name;
        this.detail = detail;
    }
} // Menu.java
```
메뉴를 구성하는 필드와 메소드가 필요하다. 그리고 각 메뉴 아이템을 담을 수 있는 생성자가 필요하다.
따라서, 각 메뉴의 번호와 이름, 설명을 지칭하는 필드를 private으로 선언하고,
private으로 선언한 각 필드에 접근 가능하도록 public getter 메소드를 선언한다.
그리고 Menu() 생성자를 통해서 메뉴 인스턴스를 생성할 수 있도록 했다.

### 2) Goods.java
```java
public class Goods extends Menu{
    private double price;     // 메뉴의 가격
    private double price2;     // 메뉴의 가격
    private double totalp;    // 메뉴 가격의 총합
    private double totalp2;    // 메뉴 가격의 총합
    private int number;   // 메뉴의 개수
    private int number2;   // 메뉴의 개수

    public int getNumber_total() { return number_total; }

    public int getNumber2_total() { return number2_total; }

    private int number_total = 0;
    private int number2_total = 0;

    public double getPrice() {
        return price;
    }              // 메뉴 가격 가져오기
    public double getPrice2() {
        this.price2 = price + 0.5;
        return price2;
    }
    @Override
    public String getName() {
        return super.getName();
    }     // 메뉴 명 가져오기 (상속)
    public String getName2() {
        return super.getName()+"(Regular)";
    }     // 메뉴 명 가져오기 (상속)
    @Override
    public String getDetail() {
        return super.getDetail();
    } // 설명 가져오기 (상속)
    @Override
    public String getNum() {
        return super.getNum();
    }       // 번호 가져오기 (상속)

    public double getTotalp() {
        this.totalp = price * number_total;
        return this.totalp;
    }    // 주문한 메뉴의 총 합을 저장하기 위한 메소드 (array에 넣어줄 값)
    public double getTotalp2() {
        this.totalp2 = price2 * number2_total;
        return this.totalp2;
    }    // 주문한 메뉴의 총 합을 저장하기 위한 메소드 (array에 넣어줄 값)

    public int getNumber() { return this.number; }               // 메뉴 개수에 접근할 메소드
    public int getNumber2() { return this.number2; }             // Regular 메뉴 개수에 접근할 메소드
    public void setNumber1(int j) {
        this.number = j;
    }           // 메뉴 개수를 수정할 메소드 (같은 메뉴를 추가하면 값을 하나씩 늘리도록)
    public void setNumber2(int i) {
        this.number2 = i; // 초기값이 이미 1이니까
    }           // 메뉴 개수를 수정할 메소드 (같은 메뉴를 추가하면 값을 하나씩 늘리도록)

    public void set_Number1(int ii) {
        this.number_total += ii;
    }

    public void set_Number2(int jj) {
        this.number2_total += jj;
    }
    // 기본 생성자
    public Goods() {}
    // 생성자
    public Goods(String num, String name, double price, String detail) {
        super(num, name, detail);
        this.price = price;
    }

} // Goods.java

```
